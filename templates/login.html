What this system demonstrates

A security-focused authentication flow that reacts to suspicious behavior in real time. Useful for showcasing defense-style monitoring and explainable risk decisions.

  • Risk scoring from behavior + signals (rapid attempts, new IP, unusual time)
  • Adaptive controls: CAPTCHA triggers + lockout decisions
  • Attack labeling: brute force / bot / credential stuffing
  • Analytics dashboard: trends, top IPs, event reasons, user count
Tip for review: Create 2–3 users, do wrong passwords, trigger CAPTCHA, then show analytics changes.
